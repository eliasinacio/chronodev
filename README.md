# React Pomofocus clone
This is a exercise to train React, Sass and Bootstrap 5. A simple clone of https://pomofocus.io.
### Goals

- [ ] Timers to 25, 5 and 15 minutes.
- [ ] Jump cycles 25 → 5 ... → 15
- [ ] Simple tasks system.
- [ ] Tasks with description.

### Secondary goals
- [ ] Save tasks locally
- [ ] Login 
- [ ] Settings
